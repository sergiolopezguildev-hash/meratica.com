This blog is part of a **100% static** site generated with [Zola](https://www.getzola.org/).

## What you will find

- Deployment and lightweight DevOps guides.
- Notes on performance, SEO and accessibility.
- Experiments with HTML, CSS and little JavaScript.

## How content is edited

1. Create a Markdown file in `content/blog/`.
2. Add front matter (title, date, taxonomies).
3. `git push` and Cloudflare runs `zola build`.

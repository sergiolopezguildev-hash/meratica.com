I'm **Meratica**. I care about software that stays maintainable, clear interfaces and architectures without unnecessary parts.

## Focus

- Prefer static when dynamic does not add value.
- Measure performance and accessibility from the first commit.
- Document enough so the project survives over time.

## Usual stack

Rust, static site generators (Zola), modern HTML/CSS, GitHub / Cloudflare Pages, and external services only when they help (for example comments with Giscus).

Soy **Meratica**. Me interesa el software que se mantiene solo, las interfaces claras y las arquitecturas sin piezas de más.

## Enfoque

- Preferir estático cuando el dinámico no aporta valor.
- Medir rendimiento y accesibilidad desde el primer commit.
- Documentar lo suficiente para que el proyecto sobreviva al tiempo.

## Stack habitual

Rust, generadores estáticos (Zola), HTML/CSS moderno, GitHub Actions / Cloudflare Pages, y servicios externos solo cuando aportan (por ejemplo, comentarios con Giscus).
